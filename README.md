# Micro Attention Transformer

An experimental transformer architecture focused on reducing attention computation through token compression using micro-attention mechanisms.

---

## Idea

Traditional self-attention scales quadratically with sequence length.

This project explores a micro-attention mechanism that compresses multiple tokens into reduced representations before applying larger attention operations.

The goal is to:
- Reduce computational complexity
- Improve scalability
- Explore efficient transformer architectures
